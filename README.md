# Username Validation in Python

This is a simple beginner Python project that validates a username using string methods and conditional statements.

## Features
- Checks maximum length (12 characters)
- No spaces allowed
- Only alphabetic characters allowed
- Displays welcome message if valid

## Concepts Used
- Python input()
- String methods (`len()`, `isalpha()`)
- Conditional statements (if-elif-else)

## How to Run
```bash
python username_validator.py
